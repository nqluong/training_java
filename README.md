# Project Structure

This project contains Java core training materials, organized by topic. Below is the folder structure under `src/`:

```
JavaCoreBasic/
├── HandleException/
│   ├── Handle Exception.docx
│   └── Handle Exception.pdf
├── Memory/
│   ├── Memory.docx
│   └── Memory.pdf
├── OOP/
│   ├── Main.java
│   ├── OOP.docx
│   ├── OOP.pdf
│   ├── interfaces/
│   │   └── Movable.java
│   └── vehicles/
│       ├── Car.java
│       ├── Motorcycle.java
│       └── Vehicle.java
├── PrimitiveAndObjectDataType/
│   ├── Main.java
│   ├── PrimitiveAndObjectDataType.docx
│   ├── PrimitiveAndObjectDataType.java
│   └── PrimitiveAndObjectDataType.pdf
├── static&final/
│   ├── static&final.docx
│   └── static&final.pdf
└── String/
    ├── Main.java
    ├── String.docx
    └── String.pdf
```

Each topic folder contains relevant Java source files and documents (DOCX, PDF) for study and reference.
